# web-font-subsetter

![Static Badge](https://img.shields.io/badge/Web%20App-wasm(run%20in%20local)-8A2BE2?style=flat-square)

このツールを使うとGoogle fontsの日本語フォントと同じ120個に分割されたサブセットを簡単に生成することができます。  
フォントファイルをドラッグアンドドロップすると、変換処理が始まります。
